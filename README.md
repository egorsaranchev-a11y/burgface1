# 🍔 BurgFace

**BurgFace** — это приложение с графическим интерфейсом для Windows, предназначенное для работы с устройствами **iPhone** и **iPad**.

Проект вдохновлён современными инструментами для управления устройствами Apple и предоставляет простой, быстрый и удобный интерфейс.

> ⚠️ BurgFace находится в активной разработке. Некоторые функции ещё не завершены и будут улучшаться в будущих версиях.

---

# ✨ Возможности

- 📱 Автоматическое обнаружение iPhone и iPad
- ℹ️ Просмотр информации об устройстве
- 🔋 Просмотр информации о батарее
- 💾 Просмотр информации о хранилище
- 📦 Создание резервных копий (Backup)
- 📋 Журнал событий (Logs)
- ⚙️ Настройки приложения
- 🌙 Светлая и тёмная тема
- 🍔 Современный интерфейс

---

# 🛠 Используемые технологии

- Python 3.12+

# 🚀 Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/USERNAME/BurgFace.git
```

Перейдите в папку проекта:

```bash
cd BurgFace
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите программу:

```bash
python main.py
```
