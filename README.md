# burgface1
🍔 BurgFace

BurgFace — это приложение с графическим интерфейсом для Windows, предназначенное для работы с устройствами iPhone и iPad. Проект вдохновлён идеей удобных утилит для управления устройствами Apple и предоставляет простой современный интерфейс.

⚠️ BurgFace находится в активной разработке. Некоторые функции ещё не завершены и будут улучшаться в будущих версиях.

Возможности
📱 Автоматическое обнаружение подключённого iPhone или iPad
ℹ️ Отображение информации об устройстве (имя, модель, версия iOS/iPadOS)
🔋 Просмотр информации о батарее (если доступна)
💾 Просмотр информации о хранилище
📦 Создание резервных копий (Backup)
📋 Встроенный журнал событий (Logs)
⚙️ Настройки приложения и смена темы
🍔 Современный интерфейс с фирменным логотипом BurgFace
Используемые технологии
Python 3.12+
CustomTkinter
Pillow (PIL)
pymobiledevice3
PyInstaller
Статус проекта

Проект находится в стадии активной разработки. Планируется добавить:

больше информации об устройстве;
улучшенную работу с резервными копиями;
дополнительные инструменты для диагностики;
более современный интерфейс;
улучшенную систему обновлений.
Установка
Клонируйте репозиторий:
git clone https://github.com/USERNAME/BurgFace.git
Установите зависимости:
pip install -r requirements.txt
Запустите приложение:
python main.py
